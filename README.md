# Analysis of Oral Transmission of Chagas Disease in Brazil (2010–2020)

This project analyzes epidemiological data from DATASUS to investigate the oral transmission of Trypanosoma cruzi in Brazil.

## Background

Chagas disease is a neglected tropical disease caused by the protozoan Trypanosoma cruzi.

Recent studies indicate that oral transmission has become the main infection pathway in Brazil.

## Objective

To analyze trends, geographic distribution and mortality patterns related to oral transmission of Chagas disease.

## Dataset

Data obtained from DATASUS public health surveillance systems.

## Tools

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings

- 2,798 confirmed cases between 2010 and 2020
- 77% related to oral transmission
- Northern region concentrates most cases
